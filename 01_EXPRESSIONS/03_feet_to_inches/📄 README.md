# Feet to Inches Converter 📏

This is a basic Python program that converts a length from **feet** to **inches** using a fixed conversion rate.

## 📐 Formula Used

1 foot = 12 inches


## 🧠 What It Does

1. Prompts the user to enter a value in **feet**.
2. Multiplies the value by 12 (since 1 foot = 12 inches).
3. Displays the result in **inches**.

## 🔁 Example Output

=== Feet to Inches Converter ===

Enter the length in feet: 5.5

5.5 feet is equal to 66.0 inches.


## 🛠 Requirements

- Python 3.x

## ▶️ How to Run

1. Save the code in a file, e.g., `feet_to_inches.py`
2. Open terminal or command prompt
3. Run the file using:

python feet_to_inches.py
